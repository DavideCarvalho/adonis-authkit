AuthKit is versioned and published with [Changesets](https://github.com/changesets/changesets).
Each published package has its own changelog and semantic version.

## GitHub releases

All tagged releases (with aggregated notes) are published here:

- [GitHub Releases](https://github.com/DavideCarvalho/streaming-educacao/releases)

## How releases work

1. A PR that changes a published package includes a changeset (`pnpm changeset`).
2. On merge to `main`, the Release workflow opens a **Version Packages** PR that
   bumps versions and updates the changelogs.
3. Merging that PR publishes the affected packages to npm — with
   [provenance](https://docs.npmjs.com/generating-provenance-statements) so the
   build origin is verifiable.
